🛡️ Vulnerability Assessment Report

📌 Overview
This repository contains a security assessment report of a public website conducted using ethical and non-intrusive methods.

🌐 Target Website
http://www.itsecgames.com

⚠️ Scope & Ethics
This assessment strictly followed ethical guidelines:

✅ Allowed
- Public pages only
- Passive scanning
- Header and configuration analysis

❌ Not Allowed
- Exploitation
- Brute force attacks
- Denial-of-Service
- Any harmful activity
  
🛠️ Tools Used

🔹 OWASP ZAP
 - Passive vulnerability scanning
 - Identified security misconfigurations

🔹 Browser Developer Tools
 - Inspected HTTP headers
 - Analyzed requests and responses

🔍 Key Findings

Vulnerability	                      Risk Level
Missing CSP                        	Medium
Missing Anti-Clickjacking Header  	Medium
Missing X-Content-Type            	Medium
Outdated Components               	Low

🎯 Conclusion
The assessment identified common security misconfigurations that may expose users to risks.
No exploitation was performed, ensuring compliance with ethical security practices.
